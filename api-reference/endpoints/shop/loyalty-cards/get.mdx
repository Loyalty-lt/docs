# Get Loyalty Card Details

Retrieve comprehensive details about a specific loyalty card, including current balance, tier status, recent transactions, and available benefits.

## Path Parameters

<ParamField path="locale" type="string" required>
  Language code for localized content (e.g., "en", "lt")
</ParamField>

<ParamField path="id" type="integer" required>
  Unique identifier of the loyalty card
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="card" type="object">
      <Expandable title="card">
        <ResponseField name="id" type="integer">
          Card identifier
        </ResponseField>
        
        <ResponseField name="card_number" type="string">
          Formatted card number
        </ResponseField>
        
        <ResponseField name="status" type="string">
          Card status (`active`, `inactive`, `suspended`)
        </ResponseField>
        
        <ResponseField name="balance" type="object">
          <Expandable title="balance">
            <ResponseField name="current_points" type="integer">
              Available points balance
            </ResponseField>
            
            <ResponseField name="tier" type="string">
              Current tier level
            </ResponseField>
            
            <ResponseField name="lifetime_earned" type="integer">
              Total points earned
            </ResponseField>
          </Expandable>
        </ResponseField>
        
        <ResponseField name="qr_code_url" type="string">
          QR code URL for card scanning
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://staging-api.loyalty.lt/en/shop/loyalty-cards/123" \
  -H "Authorization: Bearer your_jwt_token"
```
</RequestExample>
