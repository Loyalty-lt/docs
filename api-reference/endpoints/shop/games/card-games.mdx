---
title: 'Get Card Games'
api: 'GET /{locale}/shop/cards/{cardId}/games'
description: 'Get games with progress for a specific loyalty card'
---

## Overview

Get all games available for a specific loyalty card, including user progress. This is useful for displaying games on a POS when a customer's card has been scanned.

## Authentication

This endpoint requires Partner API authentication with `X-API-Key` and `X-API-Secret` headers.

## Parameters

<ParamField path="cardId" type="integer" required>
  The loyalty card ID
</ParamField>

<ParamField query="shop_id" type="integer">
  Filter games by shop ID
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of games with user progress
</ResponseField>

<ResponseField name="data[].id" type="integer">
  Game ID
</ResponseField>

<ResponseField name="data[].name" type="object">
  Localized game name
</ResponseField>

<ResponseField name="data[].game_type" type="string">
  Type of game
</ResponseField>

<ResponseField name="data[].status" type="string">
  Game status for this user: `available`, `active`, `completed`
</ResponseField>

<ResponseField name="data[].user_progress" type="object">
  User's current progress in the game
</ResponseField>

<ResponseField name="data[].user_progress.current_step" type="integer">
  Current number of stamps/progress
</ResponseField>

<ResponseField name="data[].user_progress.completed" type="boolean">
  Whether the game is completed
</ResponseField>

<ResponseField name="data[].user_progress.completed_at" type="string">
  ISO 8601 timestamp when completed
</ResponseField>

## Example Request

```bash
curl -X GET "https://staging-api.loyalty.lt/en/shop/cards/123/games?shop_id=1" \
  -H "X-API-Key: your_api_key" \
  -H "X-API-Secret: your_api_secret" \
  -H "Accept: application/json"
```

## SDK Example

```typescript
import { LoyaltySDK } from '@loyaltylt/sdk';

const sdk = new LoyaltySDK({
  apiKey: 'your_api_key',
  apiSecret: 'your_api_secret',
});

// Get games for a customer's card
const games = await sdk.getCardGames(123, 1); // cardId, shopId
console.log(games);
```
