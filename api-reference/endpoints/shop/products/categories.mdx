# Product Categories

Get the complete product category hierarchy for organizing imported products.

## Endpoint

```
GET /{locale}/shop/products/categories
```

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Partner API key
</ParamField>

<ParamField header="X-API-Secret" type="string" required>
  Partner API secret
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="categories data">
    <ResponseField name="categories" type="array">
      Hierarchical category tree (nested structure)
    </ResponseField>
    
    <ResponseField name="flat_list" type="array">
      Flat list of all categories with hierarchy indicators
    </ResponseField>
    
    <ResponseField name="total" type="integer">
      Total number of active categories
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://staging-api.loyalty.lt/en/shop/products/categories" \
  -H "X-API-Key: your_api_key" \
  -H "X-API-Secret: your_api_secret"
```

</RequestExample>
