# Points Transactions

Retrieve a paginated list of the authenticated user's points transactions, including earnings, redemptions, and bonuses.

<Info>
This endpoint requires **JWT authentication** (Bearer token). It returns transactions for the authenticated user only.
</Info>

## Path Parameters

<ParamField path="locale" type="string" required>
  Language code (e.g., `en`, `lt`)
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="per_page" type="integer" default="15">
  Items per page (max 50)
</ParamField>

<ParamField query="card_id" type="integer">
  Filter by specific loyalty card ID
</ParamField>

<ParamField query="type" type="string">
  Filter by transaction type:
  - `earned` - Points earned from purchases
  - `spent` - Points redeemed for discounts
  - `bonus` - Bonus points (signup, promotions)
  - `expired` - Expired points
  - `refund` - Refunded points
</ParamField>

<ParamField query="date_from" type="string">
  Filter transactions from date (YYYY-MM-DD)
</ParamField>

<ParamField query="date_to" type="string">
  Filter transactions to date (YYYY-MM-DD)
</ParamField>

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token from login
  
  **Format:** `Bearer eyJ0eXAiOiJKV1...`
</ParamField>

<ResponseField name="success" type="boolean">
  Request success status
</ResponseField>

    <ResponseField name="per_page" type="integer">
      Items per page
    </ResponseField>
    
    <ResponseField name="total" type="integer">
      Total transactions

    <ResponseField name="total_points_earned" type="integer">

  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://staging-api.loyalty.lt/en/shop/points?per_page=10&type=earned" \
  -H "Authorization: Bearer eyJ0eXAiOiJKV1..."
```

```javascript JavaScript
const response = await fetch(
  'https://staging-api.loyalty.lt/en/shop/points?per_page=10',
  {
    headers: {
      'Authorization': `Bearer ${accessToken}`
    }
  }
);

const data = await response.json();

console.log(`Total earned: ${data.meta.total_points_earned}`);
console.log(`Total spent: ${data.meta.total_points_spent}`);

data.data.forEach(tx => {
  const sign = tx.type === 'earned' ? '+' : '-';
  console.log(`${sign}${tx.points} - ${tx.description}`);
});
```

</RequestExample>

## Transaction Types

| Type | Description | Points |
|------|-------------|--------|
| `earned` | Points from purchases | Positive |
| `bonus` | Signup/promotional bonuses | Positive |
| `refund` | Refunded points | Positive |
| `spent` | Points redeemed | Negative |
| `expired` | Expired points | Negative |
