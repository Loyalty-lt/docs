# Get Points Balance

Retrieve a customer's current points balance. This is an alias for the [Get Card Info](/api-reference/endpoints/shop/loyalty-cards/info) endpoint, specifically for checking points.

## Quick Example

<RequestExample>

```bash cURL
curl -X GET "https://staging-api.loyalty.lt/en/shop/loyalty-cards/info?card_number=LC00000567" \
  -H "X-API-Key: your_api_key" \
  -H "X-API-Secret: your_api_secret"
```

```javascript JavaScript
async function getBalance(cardNumber) {
  const response = await fetch(
    `https://staging-api.loyalty.lt/en/shop/loyalty-cards/info?card_number=${cardNumber}`,
    {
      headers: {
        'X-API-Key': 'your_api_key',
        'X-API-Secret': 'your_api_secret'
      }
    }
  );
  
  const data = await response.json();
  return data.data.points;
}

const points = await getBalance('LC00000567');
console.log(`Current balance: ${points} points`);
```

</RequestExample>

## Query Options

| Parameter | Description |
|-----------|-------------|
| `card_number` | Loyalty card number (e.g., LC00000567) |
| `card_id` | Loyalty card ID |
| `user_id` | User ID to find their card |

<Note>
At least one query parameter is required.
</Note>
