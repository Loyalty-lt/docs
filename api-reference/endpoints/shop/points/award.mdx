# Award Points

Award loyalty points to a customer based on their purchase. This endpoint is an alias for [Create Transaction](/api-reference/endpoints/shop/transactions/create).

<Note>
For full documentation including cart items and points redemption, see [Create Transaction](/api-reference/endpoints/shop/transactions/create).
</Note>

## Quick Example

<RequestExample>

```bash cURL
curl -X POST "https://staging-api.loyalty.lt/en/shop/transactions/create" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your_api_key" \
  -H "X-API-Secret: your_api_secret" \
  -d '{
    "user_email": "customer@example.com",
    "order_id": "ORDER-12345",
    "order_total": 49.99
  }'
```

```javascript JavaScript
const response = await fetch('https://staging-api.loyalty.lt/en/shop/transactions/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key',
    'X-API-Secret': 'your_api_secret'
  },
  body: JSON.stringify({
    user_email: 'customer@example.com',
    order_id: 'ORDER-12345',
    order_total: 49.99
  })
});

const data = await response.json();
console.log(`Awarded ${data.data.points_awarded} points`);
```

</RequestExample>

<ResponseExample>

```json Success Response (200)
{
  "success": true,
  "message": "Points awarded successfully",
  "data": {

```

</ResponseExample>

## Required Fields

| Field | Type | Description |
|-------|------|-------------|
| `user_id` OR `user_email` | int/string | Customer identifier |
| `order_id` | string | Unique order reference |
| `order_total` | number | Purchase amount |

## Points Calculation

Points are automatically calculated based on your partner card settings:

```
points = (order_total / currency_amount) Ã— points_per_currency + points_per_visit
```

Override automatic calculation by providing `points` in the request.
