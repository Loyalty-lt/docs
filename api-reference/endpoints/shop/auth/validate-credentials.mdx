# Validate Credentials

Verify that your API credentials (X-API-Key and X-API-Secret) are valid and properly configured. Useful for testing integration setup.

## Endpoint

```
POST /{locale}/shop/validate-credentials
```

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Partner API key from Partners Portal (starts with `lty_`)
</ParamField>

<ParamField header="X-API-Secret" type="string" required>
  Partner API secret from Partners Portal
</ParamField>

## Path Parameters

<ParamField path="locale" type="string" required>
  Language code (e.g., `en`, `lt`)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  `true` if credentials are valid
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="valid" type="boolean">
      Whether the credentials are valid
    </ResponseField>
    
    <ResponseField name="partner" type="object">
      Partner information associated with the credentials
    </ResponseField>
    
    <ResponseField name="environment" type="string">
      Current environment (`staging` or `production`)
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST "https://staging-api.loyalty.lt/en/shop/validate-credentials" \
  -H "X-API-Key: lty_your_api_key" \
  -H "X-API-Secret: your_api_secret"
```

```javascript JavaScript
const response = await fetch('https://staging-api.loyalty.lt/en/shop/validate-credentials', {
  method: 'POST',
  headers: {
    'X-API-Key': 'lty_your_api_key',
    'X-API-Secret': 'your_api_secret'
  }
});

const data = await response.json();
if (data.success) {
  console.log('Credentials valid!', data.data.partner);
}
```

</RequestExample>

## Error Codes

| Code | Description |
|------|-------------|
| `1010` | API credentials missing |
| `1011` | Invalid API key format |
| `1012` | API key not found |
| `1013` | Invalid API secret |
| `1014` | API credentials expired |
