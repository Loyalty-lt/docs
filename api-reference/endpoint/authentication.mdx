# Authentication Endpoints

These endpoints handle user authentication, token management, and session control across the Loyalty.lt platform.

<Info>
The endpoints below are automatically generated from our OpenAPI specification. All examples include real request/response data and are interactive.
</Info>

## Endpoint Categories

### Customer Authentication
- **Login**: Authenticate customers with email/password
- **Register**: Create new customer accounts  
- **Password Reset**: Handle password recovery flow
- **Email Verification**: Verify customer email addresses

### Partner Authentication  
- **Partner Login**: Authenticate business partners
- **Staff Login**: Authenticate partner staff members
- **Role Management**: Handle partner role assignments

### Token Management
- **Token Refresh**: Refresh expired JWT tokens
- **Token Validation**: Validate token authenticity
- **Logout**: Invalidate user sessions

### Admin Authentication
- **Admin Login**: Authenticate system administrators
- **Permission Checks**: Validate admin permissions
