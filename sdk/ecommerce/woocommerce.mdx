# WooCommerce Integration

The Loyalty.lt WooCommerce plugin seamlessly integrates loyalty card functionality into your WordPress e-commerce store. Customers can earn points on purchases, redeem rewards, and track their loyalty status directly from your store.

## Plugin Overview

<Info>
The WooCommerce plugin provides a complete loyalty solution with minimal setup required. It handles point calculation, card management, and customer interactions automatically.
</Info>

### Key Features

<CardGroup cols={2}>
  <Card title="Automatic Point Awards" icon="coins">
    Points are automatically awarded when orders are completed, with configurable rates and rules.
  </Card>
  
  <Card title="Checkout Integration" icon="shopping-cart">

  </Card>
  
  <Card title="Customer Dashboard" icon="user">

## Installation

### Method 1: WordPress Plugin Directory

<Steps>
  <Step title="Access Plugin Directory">
    In your WordPress admin, navigate to **Plugins** → **Add New**.
  </Step>
  
  <Step title="Search for Plugin">
    Search for "Loyalty.lt WooCommerce" in the plugin directory.
  </Step>
  
  <Step title="Install and Activate">
    Click **Install Now**, then **Activate** once installation completes.
  </Step>
  
  <Step title="Configure Settings">
    Go to **WooCommerce** → **Loyalty Settings** to configure your API credentials.
  </Step>
</Steps>

### Method 2: Manual Installation

<Steps>
  <Step title="Download Plugin">
    Download the latest plugin ZIP file from the [Partners Portal](https://partners.loyalty.lt).
  </Step>
  
  <Step title="Upload Plugin">
    In WordPress admin, go to **Plugins** → **Add New** → **Upload Plugin**.
  </Step>
  
  <Step title="Install ZIP File">
    Choose the downloaded ZIP file and click **Install Now**.
  </Step>
  
  <Step title="Activate Plugin">
    After installation, click **Activate Plugin** to enable the loyalty features.
  </Step>
</Steps>

## Configuration

### Basic Setup

<Tabs>
  <Tab title="API Credentials">
    Configure your Loyalty.lt API credentials in **WooCommerce** → **Settings** → **Loyalty**:
    
    ```
    API Key: your_api_key_here
    API Secret: your_api_secret_here
    Environment: Staging (for testing) or Production
    ```
    
    <Warning>
    Always test with the staging environment before switching to production.
    </Warning>
  </Tab>
  
  <Tab title="Point Settings">
    Configure how points are earned and redeemed:
    
    - **Points per Dollar**: How many points customers earn per dollar spent
    - **Point Value**: Monetary value of each point when redeemed
    - **Minimum Order**: Minimum order amount to earn points
    - **Maximum Discount**: Maximum percentage of order that can be paid with points
  </Tab>
  
  <Tab title="Card Design">
    Customize the appearance of loyalty cards:
    
    - **Card Template**: Choose from available designs
    - **Welcome Points**: Initial points for new members
    - **Expiration Policy**: How long points remain valid
    - **Tier Thresholds**: Point levels for different customer tiers
  </Tab>
</Tabs>

## Customization

### Hooks and Filters

The plugin provides numerous WordPress hooks for customization:

<CodeGroup>
```php Point Calculation
// Customize point calculation
add_filter('loyalty_lt_calculate_points', function($points, $order, $customer) {
    // Double points for VIP customers
    if ($customer->has_role('vip')) {
        $points *= 2;
    }
    
    // Bonus points for large orders
    if ($order->get_total() > 200) {
        $points += 50;
    }
    
    return $points;
}, 10, 3);
```

```php Custom Rewards
// Add custom redemption options
add_action('loyalty_lt_checkout_rewards', function($customer_points) {
    if ($customer_points >= 500) {
        echo '<div class="custom-reward">';
        echo '<input type="checkbox" id="free_shipping" name="loyalty_rewards[]" value="free_shipping">';
        echo '<label for="free_shipping">Free Shipping (500 points)</label>';
        echo '</div>';
    }
});
```

```php Email Customization
// Customize loyalty emails
add_filter('loyalty_lt_email_content', function($content, $type, $customer) {
    if ($type === 'points_earned') {
        $content .= "\n\nVisit our rewards catalog to see what you can redeem!";
    }
    return $content;
}, 10, 3);
```
</CodeGroup>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Points Not Awarding">
    **Symptoms**: Customers complete orders but don't receive points
    
    **Solutions**:
    - Check API credentials in plugin settings
    - Verify order status is set to "Completed"
    - Ensure customer has a loyalty card
    - Check product/category exclusion rules
    - Review minimum order amount settings
  </Accordion>
  
  <Accordion title="Checkout Integration Issues">
    **Symptoms**: Point redemption not working during checkout
    
    **Solutions**:
    - Clear cache if using caching plugins
    - Check for JavaScript conflicts in browser console
    - Verify AJAX requests are reaching the server
    - Test with default theme to isolate theme conflicts
    - Ensure WooCommerce is up to date
  </Accordion>
  
  <Accordion title="API Connection Problems">
    **Symptoms**: Plugin shows connection errors
    
    **Solutions**:
    - Verify API credentials are correct
    - Check if staging/production environment is correct
    - Ensure server can make outbound HTTPS requests
    - Check firewall settings on hosting provider
    - Test API connection using staging environment
  </Accordion>
  
  <Accordion title="Customer Data Sync Issues">
    **Symptoms**: Customer loyalty data not matching between systems
    
    **Solutions**:
    - Trigger manual sync from plugin settings
    - Check for duplicate customer accounts
    - Verify email addresses match exactly
    - Review customer merge/deletion logs
    - Contact support for data reconciliation
  </Accordion>
</AccordionGroup>

### API Request Optimization

<Tabs>
  <Tab title="Batch Operations">
    ```php
    // Batch multiple point awards
    $batch_transactions = [];
    foreach ($orders as $order) {
        $batch_transactions[] = [
            'card_id' => $order->get_customer_loyalty_id(),
            'points' => calculate_points($order),
            'reference' => $order->get_id()
        ];
    }
    
    // Single API call for all transactions
    $loyalty_api->bulk_award_points($batch_transactions);
    ```
  </Tab>
  
  <Tab title="Request Queuing">
    ```php
    // Queue API requests for background processing
    wp_schedule_single_event(time() + 60, 'loyalty_process_queue', [
        'action' => 'award_points',
        'data' => $point_data
    ]);
    ```
  </Tab>
  
  <Tab title="Error Handling">
    ```php
    // Implement retry logic for failed requests
    function retry_api_request($callback, $max_attempts = 3) {
        for ($i = 0; $i < $max_attempts; $i++) {
            try {
                return $callback();
            } catch (Exception $e) {
                if ($i === $max_attempts - 1) throw $e;
                sleep(pow(2, $i)); // Exponential backoff
            }
        }
    }
    ```
  </Tab>
</Tabs>

### API Security

<CardGroup cols={2}>
  <Card title="Credential Storage" icon="key">

  <Card title="Request Validation" icon="shield">

    - Sanitize point amounts and references
    - Check user permissions before API calls
    - Implement rate limiting for API requests

  <Card title="Data Protection" icon="lock">

    - Implement proper session management

  <Card title="Access Control" icon="user-shield">
